## Czech Alphabet Coverage

All Czech letters with diacritics are accessible:

- **With Caron (háček):** č, ď, ě, ň, ř, š, ť, ž
  - Direct: `č` `ě` `ň` `ř` `š` `ž`
  - With dead key (=+Level3): `ď` `ť`
  
- **With Acute (čárka):** á, é, í, ó, ú, ý
  - Direct: `á` `é` `í` `ú` `ý`
  - Standard: `ó` (type `o` normally, not common)
  
- **With Ring (kroužek):** ů
  - Direct: `ů`

## Tips

1. **Capitals:** Use Caps Lock or Shift as normal: `Č` `Ř` `Ů` etc.
2. **Numbers:** Hold Level3 key (ISO extra key) and press number row
3. **Dead Keys:** 
   - Level3+Equals, then letter = add caron to any letter
4. **Muscle Memory:** All US symbol positions unchanged when using Shift
